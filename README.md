# ⚡ CYD Captive Portal

> *“In a world where everything is connected, control is everything.”*

---

## 🧠 Overview

This project is a **custom captive portal** designed for the **Cheap Yellow Display (CYD / ESP32)**.

When a device connects to the network, it is redirected to a **custom portal** featuring:
- Animated terminal sequence
- Fake network scanning + payload injection
- Simulated progress bars
- Final message screen with a privacy-focused quote

Built to be:
- ⚡ Lightweight  
- 📱 Mobile-compatible (works on captive portal browsers)  
- 🔒 Non-malicious (visual/demo purposes only)  

---

## 🎯 Features

- 💻 Terminal-style connection simulation  
- 📡 Fake IP + device detection  
- 📊 ASCII-based progress bar (mobile safe)   
- 🧬 Smooth transition to final message screen  
- 📱 Fully responsive (desktop + mobile)  
- ⚙️ No JavaScript required (maximum compatibility)  

---

## 📂 Setup

1. Format your SD card to **FAT32**
2. Create a file named: index.html
3. paste the script of your choice into index.html
4. Ensure index.html is in root / it will not work if its not
